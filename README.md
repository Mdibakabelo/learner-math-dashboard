# 📊 Learner Mathematics Improvement Dashboard (Power BI)

## 📌 Project Overview

This project showcases the academic progress of learners I have been tutoring over a 5-year period.

The goal of the dashboard is to track Mathematics performance from Grade 4 to Grade 8, using data to highlight improvement trends across school terms and academic years.

## 🎯 Objective

The main objective of this project is to demonstrate how data analysis can be used to:

* Monitor learner performance over time
* Identify improvement trends
* Measure the impact of consistent tutoring

## 📂 Dataset

The dataset was created to simulate real tutoring outcomes and includes:

* 5 learners
* 5 years of data (2020–2024)
* 4 school terms per year
* Mathematics scores (%)

## 📈 Key Insights

* All 5 learners improved from approximately 40% to above 80%
* There is a consistent upward trend in performance across all learners
* Term-based analysis shows steady academic growth
* The data reflects the positive impact of structured and continuous tutoring

## 📊 Dashboard Features

* Interactive Power BI dashboard
* Learner performance tracking over time
* Yearly and term-based analysis
* KPI indicators such as average score and improvement percentage
* Filters for learner, year, and grade

## 🛠 Tools Used

* Power BI
* Microsoft Excel

## 🚀 How to Use

1. Download the dataset
2. Open Power BI Desktop
3. Load the dataset
4. Explore the interactive dashboard
